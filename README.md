# React Mini Projects Portfolio

This repository contains frontend mini projects built with React concepts. It includes both CDN-based static apps and JSX component source files.

## Repository
- GitHub: https://github.com/AshishCherian15/react-mini-projects

## Projects Overview

| Project | Folder | Primary Files | Current Live Link |
|---|---|---|---|
| Counter App | `Counter/` | `counter-app.html` | Add after deployment |
| Expense Tracker | `Expense/` | `ExpenseTracker.jsx`, `index.html` | Add after deployment |
| Profile Card | `Profile-card/` | `ProfileCard.html` | Add after deployment |
| Todo List | `TO DO/` | `TodoApp.jsx`, `index.html` | Add after deployment |

## Run Projects Locally

### Counter
- Open `Counter/counter-app.html` in your browser.

### Profile Card
- Open `Profile-card/ProfileCard.html` in your browser.

### Expense Tracker
- Open `Expense/index.html` in your browser (website-style version).
- Or use `Expense/ExpenseTracker.jsx` inside a React app as `App.jsx`.

### Todo List
- Open `TO DO/index.html` in your browser (website-style version).
- Or use `TO DO/TodoApp.jsx` inside a React app as `App.jsx`.
